QUICK REFERENCE — CIPHERS & ENCODING
Caesar Cipher
Shift every letter forward or backward by a fixed number. Shift 3 means A becomes D, B becomes E, Z wraps to C.

To decode: shift backward by the same number.
Example: KHOOR (shift 3 back) = HELLO
ROT13
A Caesar cipher with shift exactly 13. Because the alphabet has 26 letters, applying ROT13 twice returns the original. A=N, B=O, C=P...
A-MN-Z
N-ZA-M
Example: URYYB (ROT13) = HELLO
Binary (ASCII)
Each 8-bit group is a number. Convert to decimal, then look up the ASCII character.
01001000 = 72 = H   01001001 = 73 = I
Hex (Base 16)
Uses digits 0–9 then A–F. Two hex digits = one byte = one ASCII character.
48 = H   65 = e   6C = l   6F = o
